JavaScript (JS) basics

Gonna start with learning JS comments, console.log, window.alert, and document.getElementById.

With console.log I can output text to the the console. This is mainly used for debugging and the result can be seen in the browser dev tools.


console.log("Hello World!");

With window.alert I can display an alert on the browser window


window.alert("This is an alert!");

The document.getElementById select an HTML element on the page by using it's id. Then you can manipulate that element with JS.


document.getElementById("text").textContent = "Hello!";
